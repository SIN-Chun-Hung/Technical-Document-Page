Variable

Variables are powerful in Python. They help storing information (in other term: data) and performing several operations such as while loop controls and update the calculation result. Information/data stored in the variable is regarded as value. Any data types of Python such as integer float, string, list, tuple, dictionary can be stored in the variable as value.

Create Variable

While creating variable, there are some rules that you should strictly follow. Otherwise, variable cannot be created and some errors appear regarding the name of it. All the rules are shown below.

  • Variable name can be in short such as x, y, z
  • Variable name can be in descriptive form such as my_favourite_subject, the_3_mathematicians, Python_is_great_and_amazing
  • A variable name must start with lowercase character, uppercase character or underscore character_
    • . It CANNOT start with a number e.g. 3.
  • A variable name can only contain lower, upper letters and underscore character. a-z, A-Z, _
  • A variable name is case-sensitive. math, matH, Math, MaTh, MATH are all different variables.

Apart from the rules of creation, the step of defining variable is to start with creating a variable name, then use an assignment operator = connect the value you would like to store. For example,

MyName = "HUNG"

His_univ = "HKUST"
which store the strings into the two variable respectively.

You can also reassign a value to a variable but the old value within that variable will be totally replaced. For example, reassign my name,

MyName = "SIMON"
"HUNG" will be replaced by "SIMON".

One single line statement can be used to define multiple variables. For example,

MyName, What_University, Major = "HUNG", "HKUST", "Mathematics"
This one line stores "HUNG", "HKUST" and "Mathematics" into corresponding variables.

Multiple variables can also point to the same value and store the same value in one line statement. For example,

MyName1 = MyName2 = MyName3 = "SIMON"
This one line code use the assignment operator as a chain, connecting three variables to the same value string "SIMON".

In many cases, update of a variable is crucial to operation of computer programming. This even plays an important role in developing different kinds of mathematical and computer model to solve and analyze real world problem. Therefore, it is an essential skill to learn. Apart from pointing the variable to a new value, the original old value within it itself can be used to update by combining with some operators. For example,

number = 10

number = number operator 7

print(number)
The above block of code has the following meaning: First define number is equal to 10. Then use this number combine a operator, can be +, -, *, **, /. After the operator, the 7 will be combined with the old value number 10, returning the final number as a result storing in the number variable. The final value shows in the print statement in the final line of code.

If Else Statement

Programming language has great and powerful features to do branching. Branching means that a block of code statements, which is associated with one or more conditions in the same if line or else line or elif line are evaluated to be true at the same time, will be executed. If there is found that more than one block of statements' conditions are true, the first block of code statement will be executed and other will be ignored.

There can be several combinations of if, elif, else. The following will show couple of examples.

if condition1:

block of statements within if

else:

block of statements in else

In the first example, if Python evaluate the condition1 to be True, then the block of statements within if will be executed. If the condition1 is False, then the block of statements in else will be executed instead of the first block. Apart from this principle, all the blocks with theirs code line statements are indented 4 spaces such that the program will be easier to read and meet the syntax requirement of Python. The next will show the second example.

if condition1:

block of statements within if

elif condition2:

block of statements in elif

else:

block of statements in else

In the second example, if the condition1 is True, then the block associated with if will be executed, the other two will be ignored automatically. If the condition2 is True, other is false, then the block within elif will be executed. Another case is that if all condition1 and condition2 seems to be False, the block within else will be executed. Final situation introduced here is condition1 and condition2 are True, then the first block which starts from the top to the bottom will be prioritized and executed, then escape from this if..elif..else statement and run other codes. Below is the third and final example.

if condition1 and condition2:

block of statements within if

elif condition3 or condition4:

block of statements in elif

elif condition5:

block of statements in elif

else:

block of statements in else

In this third and final example, if condition1 and condition2 both are correct at the same time, then the block of statements of code within if will be executed. However, and indicates both condition must be True. If one of them is False, the whole expression i.e. condition1 and condition2 is equal to False, the whole block within if will not be executed and jump to the next elif block to evaluate other conditions.or indicates one of the conditions must be True such that the whole condition3 or condition4 is equal to True. If condition3 or condition4 is True, then the block within that elif will be executed. If condition3 and condition4 are False, the block will not be executed and jump to the next block. If condition5 is True, then return the result of the block of codes within that elif. If condition5 is False, jump to the else block and execute that block of codes.

While Statement

In programming, there exists occassions that block of codes need to be executed repeatedly. In order to repeats codes without copying them several times in the IDE or editor, while statement will do the job with the use of condition as if statement. The following is the structure of while loop:

while condition:

block of codes

where the block are indented by 4 spaces as in the if statement.

Infinite Loop

The following example is an infinite loop:

while condition:

block of codes

As the condition is True and no constraints are applied to turn the condition to False, then the block of statement will run forever. Infinite number of results will be returned. The next example will apply those constraints.

Using Variable

This example will define a variable and use as the condition of while statement.

i = 1

while i < 21:

block of codes

i = i + 1

The block of codes will be executed 20 times because the variable i will be increased by 1 when the block is executed one time. The last execution is when i = 20, the block of code executes along with adding 1 to i, becoming i = 21 and turning the while condition from True to False. As a result, the while loop is broken and run the codes below this while loop. The next and the final example is the introduction for break command.

Using Break Command

This is the last example with the use of break command.

while condition:

block of codes

break

In this example, the block of codes will not be executed infinitely but to be interupted. The condition is True, so the block of codes are executed. After running this block of codes, Python will arrive the line break just below the final line of the block of codes. This break will immediately stop the while loop and straightly jump to the remaining part of the codes below the while loop.
